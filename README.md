# Jääkiekkotilasto

Tietokantasovellus jääkiekko-ottelun ottelutapahtumien kirjaamista ja selaamista varten. Ottelutapahtumista tilastoidaan maalit ja rangaistukset. Käyttäjä voi tarkastella tarvittaessa pelaajakohtaisia ja joukkuekohtaisia tilastoja. Esimerkiksi pelaajan tekemät maalit ja syötöt yhteensä tai joukkueen tekemät maalit yhteensä.
